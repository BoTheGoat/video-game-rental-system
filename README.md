# Video Game Rental System 🎮

A **console-based backend application** built with **Java** and **SQL** to manage video game rentals. This project demonstrates object-oriented programming, database integration with JDBC, and basic backend logic handling.

---

## Features

- Add and manage **users**  
- Add and manage **video games**  
- Rent and return games  
- View all available games  
- Track rental history and inventory  

---

## Technologies Used

- **Java** – Core application logic  
- **SQL** – Database management  
- **JDBC** – Java database connectivity
